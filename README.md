# Electronics Network API

API-платформа для управления сетью по продаже электроники.

## Стек
- Python
- Django
- Django REST Framework
- PostgreSQL
- Docker / Docker Compose
- JWT
- drf-spectacular

## Функциональность
- Управление звеньями сети
- Иерархия поставщиков
- Фильтрация по стране
- JWT-аутентификация
- Админ-панель
- Swagger/OpenAPI

## Запуск проекта

```bash
docker compose up --build
```
